## 常见问题排查

### Q1: 登录失败，提示"用户名或密码错误"

**可能原因**：
- 数据库中不存在该用户
- 密码 MD5 加密后不匹配

**解决方法**：
```bash
# 检查数据库中的用户
docker exec absadmin-mysql mysql -uroot -proot123 absframe -e "SELECT loginname FROM abs_userinfo LIMIT 5;"
```

---

### Q2: AI 助手回复失败

**可能原因**：
- DeepSeek API Key 未正确配置
- 网络连接问题
- 后端服务未运行

**排查步骤**：
```bash
# 1. 检查容器中是否有 API Key
docker exec absadmin-app env | grep DEEPSEEK

# 2. 检查后端日志
docker logs absadmin-app --tail 50

# 3. 测试 API 连通性
curl -X POST http://localhost:8020/ai/create \
  -H "Content-Type: application/json" \
  -d '{"type":"text","topic":"你好","style":"professional","length":"medium","keywords":"","model":"deepseek"}'
```

---

### Q3: 班级管理数据加载失败

**可能原因**：
- 数据库中 base_class 表为空
- API 路径不正确

**排查步骤**：
```bash
# 1. 检查数据库表
docker exec absadmin-mysql mysql -uroot -proot123 absframe -e "SELECT * FROM base_class LIMIT 5;"

# 2. 检查浏览器 Network 面板
# 查看请求路径是否为：POST /class/getpagelist
# 查看响应内容
```

---

### Q4: 页面显示空白

**可能原因**：
- 前端路由配置错误
- 组件文件不存在
- 浏览器控制台有 JavaScript 错误

**解决方法**：
1. 打开浏览器开发者工具（F12）
2. 查看 Console 标签是否有错误
3. 查看 Network 标签是否有 404 错误
4. 根据错误信息定位问题
